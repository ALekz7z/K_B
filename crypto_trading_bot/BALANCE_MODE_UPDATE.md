# Обновление: Автоматическое переключение SPOT/FUTURES режимов

## 📋 Обзор изменений

Бот теперь **автоматически переключается** между режимами торговли в зависимости от баланса:

| Баланс | Режим | Плечо | Размер позиции | Стоп-лосс | Шорты |
|--------|-------|-------|----------------|-----------|-------|
| **< 500 USDT** | **SPOT** | 1x (без плеча) | 12% от депозита | 4% (мягче) | ❌ Запрещены |
| **≥ 500 USDT** | **FUTURES** | 10x (настраиваемое) | 2.5% риска | 2% (строгий) | ✅ Разрешены |

---

## 🔧 Новые файлы

### `modules/risk_manager.py`
Центральный модуль управления рисками с функциями:
- ✅ Автоматическое определение режима торговли (SPOT/FUTURES)
- ✅ Расчет размера позиции с учетом плеча
- ✅ Управление стоп-лоссами и тейк-профитами
- ✅ Отслеживание серии убытков
- ✅ Контроль одновременных позиций
- ✅ Trailing stop (скользящий стоп)
- ✅ Тайм-ауты для позиций без движения

---

## 📊 Примеры расчета позиций

### При балансе 100 USDT (SPOT режим):
```
Цена BTC: 50,000 USDT
Размер позиции: 0.001 BTC
Стоимость: 50 USDT (12% от баланса)
Плечо: 1x
Стоп-лосс: 4% = 48,000 USDT
Тейк-профит 1: +1.5% = 50,750 USDT
Тейк-профит 2: +3.0% = 51,500 USDT
```

### При балансе 600 USDT (FUTURES режим):
```
Цена BTC: 50,000 USDT
Размер позиции: 0.003 BTC
Стоимость: 150 USDT (2.5% × 10x плечо)
Плечо: 10x
Стоп-лосс: 2% = 49,000 USDT
Тейк-профит 1: +2 USDT прибыли
Тейк-профит 2: +3.5 USDT прибыли
Trailing stop: после +1.5 USDT → в безубыток
```

## ⚙️ Настройки в `config/settings.py`

### Порог переключения режимов:
```python
BALANCE_THRESHOLD_USDT = 500  # При балансе >= 500 → FUTURES
```

### Параметры SPOT режима:
```python
SPOT_POSITION_SIZE_PERCENT = 12.0  # 12% от баланса в сделку
SPOT_LEVERAGE = 1  # Без плеча
SPOT_STOP_LOSS_PERCENT = 4.0  # Мягкий стоп-лосс
SPOT_TAKE_PROFIT_1_PERCENT = 1.5  # Фиксация 50% при +1.5%
SPOT_TAKE_PROFIT_2_PERCENT = 3.0  # Закрытие при +3%
SPOT_ALLOW_SHORT = False  # Шорты запрещены
```

### Параметры FUTURES режима:
```python
FUTURES_POSITION_SIZE_PERCENT = 2.5  # 2.5% риска на сделку
FUTURES_LEVERAGE = 10  # 10x плечо
FUTURES_STOP_LOSS_PERCENT = 2.0  # Строгий стоп-лосс
FUTURES_TAKE_PROFIT_1_USDT = 2.0  # Фиксация 50% при +2 USDT
FUTURES_TAKE_PROFIT_2_USDT = 3.5  # Закрытие при +3.5 USDT
FUTURES_TRAILING_STOP_ACTIVATION_USDT = 1.5  # Trailing stop после +1.5 USDT
```

## 🛡️ Механизмы защиты

### Для SPOT режима (малый депозит):
- ✅ Увеличенный размер позиции (12%) для роста без плеча
- ✅ Мягкий стоп-лосс (4%) против волатильности
- ✅ Запрет шортов (только лонги)
- ✅ Меньше одновременных сделок (макс 2)
- ✅ Более длинный тайм-аут (15 мин)

### Для FUTURES режима (депозит ≥ 500 USDT):
- ✅ Ограниченное плечо (10x по умолчанию, макс 20x)
- ✅ Строгий риск-менеджмент (2% на сделку)
- ✅ Trailing stop для защиты прибыли
- ✅ Все стратегии доступны (LONG/SHORT/RANGE)
- ✅ Быстрые тайм-ауты (7 мин)

---

## 🎯 Стратегия распределения капитала

### Если у вас 100 USDT:
1. Бот торгует на **SPOT** без плеча
2. Размер позиции: **12 USDT** за сделку (12%)
3. Можно открыть **2 позиции** одновременно (24 USDT)
4. Цель: накопить до 500 USDT

### Если у вас 500+ USDT:
1. Бот переключается на **FUTURES** с плечом 10x
2. Риск на сделку: **2.5%** от баланса
3. Эффективная позиция: **25% от баланса** (2.5% × 10x)
4. Доступны все стратегии включая шорты

---

## 📝 Условия закрытия сделок

### Общие для обоих режимов:
1. **Стоп-лосс** – принудительное закрытие при убытке
2. **Тейк-профит 1** – закрытие 50% позиции
3. **Тейк-профит 2** – закрытие оставшихся 50%
4. **Trailing Stop** – перенос стопа в безубыток
5. **Тайм-аут** – закрытие при отсутствии движения
6. **Серия убытков** – пауза после 3-5 неудачных сделок
7. **Смена фазы рынка** – закрытие при изменении тренда
