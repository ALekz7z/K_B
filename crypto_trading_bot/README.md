# Crypto Trading Bot для Bybit

## Полная стратегия торговых ботов для криптовалютной торговли

### 📋 Описание

Это комплексная система автоматической торговли криптовалютами на бирже Bybit, реализующая стратегию скальпинга с целевой прибылью 3-4 USDT за сделку.


### 🔧 Модули системы

#### 1. Market Analyzer (Аналитик)
- Определение фазы рынка (LONG, SHORT, RANGE, UNCERTAIN)
- Выбор топ-3 монет для торговли по 6 фильтрам:
  - Ликвидность (объем > 50M USDT, спред < 0.1%)
  - Волатильность (изменение 3-8%, ATR > 0.5%)
  - Корреляция с BTC (0.3-0.8)
  - Сентимент-анализ
  - Анализ ордербука
  - Технические условия (RSI 30-70, Bollinger Bands)
- Адаптивные параметры (машинное обучение)

#### 2. Long Trader (Бот-лонгист)
**Стратегии:**
1. Пробой сопротивления
2. Отскок от поддержки
3. Скальпинг на волатильности

#### 3. Short Trader (Бот-шортист)
**Стратегии:**
1. Пробой поддержки
2. Отскок от сопротивления
3. Скальпинг на перекупленности

#### 4. Range Trader (Бот-боковик)
**Стратегии:**
1. Торговля в диапазоне
2. Mean Reversion (возврат к среднему)
3. Паттернное скальпинг

### ⚙️ Параметры системы

#### Торговые параметры
- Размер позиции: 2-3% от депозита
- Целевая прибыль: 3-4 USDT за сделку
- Стоп-лосс: 1-2% от цены входа
- Частичная фиксация: 2 USDT (50% позиции)
- Полная фиксация: 3-4 USDT (50% позиции)
- Скользящий стоп: после +1.5 USDT → безубыток
- Тайм-аут: 5-10 минут без движения

#### Индикаторы
- EMA: 50, 100, 200 (адаптивный диапазон 40-60)
- ADX: период 14, порог 25 (адаптивный 20-30)
- MACD: 12, 26, 9
- RSI: период 14, границы 30-70
- Bollinger Bands: 20 периодов, 2 стандартных отклонения
- ATR: период 14

#### Риск-менеджмент
- Максимум 3 открытые сделки одновременно
- Максимум 2 разные монеты одновременно
- Общий риск: не более 6% от депозита
- Стоп после 3-5 убыточных сделок подряд

### 🚀 Запуск

1. **Установка зависимостей:**
```bash
pip install -r requirements.txt
```

2. **Настройка API ключей:**
Отредактируйте `config/settings.py`:
```python
BYBIT_API_KEY = "your_api_key"
BYBIT_API_SECRET = "your_api_secret"
BYBIT_TESTNET = True  # Используйте тестовую сеть для начала
```

3. **Запуск бота:**
```bash
cd crypto_trading_bot
python main.py
```

### 📊 Мониторинг и статистика

Система отслеживает:
- Текущую фазу рынка
- Выбранные монеты и их рейтинги
- Открытые позиции (PnL в реальном времени)
- Статистику сделок (win rate, средняя прибыль/убыток)
- Серии убытков

### ⚠️ Важные замечания

1. **Тестирование:**
   - Сначала протестируйте на исторических данных (бэктест)
   - Затем используйте демо-счет (Bybit Testnet)
   - Только потом запускайте на реальных деньгах

2. **Комиссии:**
   - Maker: ~0.02%
   - Taker: ~0.05%
   - Учтены в расчетах прибыли

3. **Безопасность:**
   - Храните API ключи в защищенном месте
   - Используйте ключи только с правами на торговлю (без вывода)

4. **Обработка ошибок:**
   - Все API вызовы имеют обработку ошибок
   - Реализованы повторные попытки при сбоях
   - Полное логирование всех действий

### 📈 Цикл работы

1. **Каждые 15 минут:** Анализ рынка и выбор монет
2. **Постоянно:** Проверка условий входа по стратегиям
3. **Постоянно:** Управление открытыми позициями
4. **Постоянно:** Контроль рисков
5. **Каждые 10 секунд:** Обновление мониторинга

### 🛑 Условия остановки

- Достигнут порог убыточных сделок (3-5 подряд)
- Смена фазы рынка аналитиком
- Найдена более подходящая монета
- Превышено время торговли 4 часа без перерыва
