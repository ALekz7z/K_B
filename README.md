# MEXC Trading Bot - Cryptocurrency Market Making Bot

[![Python 3.9+](https://img.shields.io/badge/python-3.9+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Асинхронный торговый бот для биржи MEXC Global, реализующий стратегию сбора спреда (Market Making) на неликвидных спотовых парах.

## ⚠️ Важное предупреждение

**Этот бот предназначен для образовательных целей. Используйте его на свой страх и риск.**

- Начните с режима **Paper Trading** (бумажная торговля)
- Никогда не используйте деньги, которые вы не готовы потерять
- Проверяйте код перед запуском на реальные средства
- API-ключи должны иметь права **ТОЛЬКО на Spot Trading** (никогда не включайте Withdrawal!)

## 🎯 Возможности

- **Асинхронная архитектура** - полная поддержка asyncio для высокой производительности
- **Market Making стратегия** - автоматический сбор спреда на неликвидных парах
- **Режим Paper Trading** - безопасное тестирование без реальных денег
- **Управление рисками**:
  - Дневной лимит убытка ($5 по умолчанию)
  - Максимальный размер позиции ($10 по умолчанию)
  - Стоп-лосс 2%
  - Проверка баланса
  - Лимит одновременных позиций (2)
- **Сканирование рынка**:
  - Исключение топ-20 монет по объёму
  - Фильтр по минимальному объёму ($50,000+)
  - Избегание пампов/дампов (>15% за час)
  - Минимальный спред 1.5%
- **Логирование** - подробные логи в консоль и файл
- **История сделок** - запись всех сделок в CSV
- **Защита от ошибок** - обработка API ошибок, сетевых проблем

## 🔐 Настройка

### Шаг 1: Создание файла .env

```bash
cp .env.example .env
```

### Шаг 2: Получение API-ключей MEXC

1. Зайдите на [MEXC.com](https://www.mexc.com/)
2. Войдите в аккаунт
3. Перейдите в **Account → API Management**
4. Нажмите **Create API Key**
5. Пройдите верификацию (2FA, email)
6. **ВАЖНО:** При создании ключей:
   - ✅ Включите **Spot Trading**
   - ❌ **ОТКЛЮЧИТЕ Withdrawal** (вывод средств)
   - ❌ **ОТКЛЮЧИТЕ Futures Trading** (если не используется)
7. Сохраните API Key и Secret Key

### Шаг 3: Настройка .env

Откройте файл `.env` и заполните:

```env
# Ваши API ключи от MEXC
MEXC_API_KEY=ваш_api_key_здесь
MEXC_SECRET_KEY=ваш_secret_key_здесь

# Режим торговли (True = безопасный режим, False = реальные деньги)
PAPER_TRADING=True

# Лимиты (не меняйте без понимания рисков!)
MAX_POSITION_SIZE=10
MAX_DAILY_LOSS=5
```

### Шаг 4: Проверка конфигурации

```bash
python -c "from config import Config; print('Config OK:', Config.validate())"
```

## 📊 Стратегия

### Как работает бот

1. **Сканирование рынка**
   - Получает все USDT пары
   - Исключает топ-20 монет (BTC, ETH, SOL...)
   - Фильтрует по объёму (>$50k за 24ч)
   - Ищет пары со спредом >1.5%

2. **Анализ стакана**
   - Проверяет ликвидность на лучших ценах
   - Минимум $10 на bid и ask

3. **Вход в сделку**
   - Buy Price = Best Bid × 1.001 (на 0.1% выше лучшего спроса)
   - Sell Price = Best Ask × 0.999 (на 0.1% ниже предложения)
   - Размер позиции: максимум $10 (10% от депозита)

4. **Выход из сделки**
   - **Цель**: продажа по Sell_Price
   - **Тайм-аут**: 10 минут → продажа по рынку
   - **Стоп-лосс**: -2% от цены покупки → немедленная продажа

5. **Цикл**
   - Пауза 5 секунд
   - Возврат к шагу 1
   - Максимум 2 одновременные позиции

## 🛡️ Управление рисками

| Параметр | Значение | Описание |
|----------|----------|----------|
| MAX_POSITION_SIZE | $10 | Макс. размер одной позиции |
| MAX_DAILY_LOSS | $5 | Макс. убыток за день |
| STOP_LOSS_PERCENT | 2% | Автоматический стоп-лосс |
| MAX_CONCURRENT_POSITIONS | 2 | Одновременных сделок |
| MIN_BALANCE | $5 | Мин. баланс для работы |
| ORDER_TIMEOUT | 10 мин | Тайм-аут ордера |

### Защита от потери средств

1. **Дневной лимит убытка** — при достижении $5 убытка бот останавливается до 00:00 UTC
2. **Стоп-лосс 2%** — автоматическая продажа при падении цены
3. **Проверка баланса** — остановка при балансе <$5
4. **Тайм-аут соединения** — отмена ордеров при потере связи >1 минуты
5. **No Martingale** — бот никогда не усредняет убыточные позиции

## ❓ FAQ

### Q: Безопасно ли это?
**A:** Бот включает множество защитных механизмов, но криптовалютная торговля всегда рискованна. Начинайте с Paper Trading!

### Q: Сколько нужно денег для старта?
**A:** Рекомендуется минимум $100. Бот настроен на максимальную позицию $10 (10% от депозита).

### Q: Можно ли запустить на Windows?
**A:** Да, но рекомендуется Linux VPS для стабильной круглосуточной работы.

### Q: Как часто бот торгует?
**A:** Зависит от рыночных условий. На неликвидных парах может быть 5-20 сделок в день.

### Q: Что если бот зависнет?
**A:** Реализована защита от зависания — при потере соединения >1 минуты все ордера отменяются.

### Q: Нужно ли держать бота включённым постоянно?
**A:** Да, для работы стратегии бот должен работать непрерывно.

### Q: Какие комиссии?
**A:** На MEXC спотовые комиссии часто 0%. В коде есть параметр FEE_PERCENT для учёта комиссий.

## 🔧 Troubleshooting

### Ошибка "Authentication failed"
- Проверьте API-ключи в .env
- Убедитесь, что ключи активны в кабинете MEXC
- Проверьте, что включены права Spot Trading

### Ошибка "Insufficient funds"
- Проверьте баланс USDT
- Уменьшите MAX_POSITION_SIZE
- Убедитесь, что нет замороженных средств в других ордерах

### Бот не находит пары для торговли
- Увеличьте MIN_SPREAD_PERCENT в .env
- Уменьшите MIN_24H_VOLUME
- Проверьте, что исключены не все монеты

### Частые rate limit ошибки
- Увеличьте POLLING_INTERVAL
- Уменьшите количество проверяемых пар
